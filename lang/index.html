Contribute Support for your Language

Probably the easiest way to contribute a new translation to DS4Windows is to edit and submit one on Transifex. Another option is to fork the DS4Windows project on GitHub, copy Strings.resx and use a filename relevant to the language you wish to translate (DS4Windows\Translations\Strings..resx). Edit the values and submit a pull request on GitHub.

About DS4Windows

This fork of DS4Windows is a project originally based on the work of Jays2Kings. Many changes have been made to the project since it was originally forked in March 2017. The head of the current project is Ryochan7 and many contributions have been made on GitHub by users.
